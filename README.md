# SYS-265 Repository

This repository contains configuration files, scripts, and documentation for SYS-265.

## Directory Structure

- `SYS265/linux/` - Linux configuration files and scripts
- `SYS265/windows/` - Windows configuration files and scripts
- `wiki/` - Course tech journal entries

## Systems

- docker01: Ubuntu Docker host
- web01: Linux web server
- mgmt01: Windows management server
